Multi-Role Store Rating Web App

• (Built with Next.js, PostgreSQL, Prisma, Tailwind, JWT, Zod, React Hook Form)
• Built a full-stack web application using Next.js App Router for users to register, login, create stores, and submit ratings.
• Implemented role-based access control for Admin, User, and Store Owner using JWT authentication and custom middleware.
• Designed a responsive UI with Tailwind CSS
• Integrated PostgreSQL with Prisma ORM, handling complex relationships like users, stores, and nested ratings.
• Developed dynamic dashboards for Admins (user/store stats, filters, search), and Store Owners (store management and analytics).
• Deployed to Vercel, configured .env secrets and secure database, and followed scalable folder architecture with src/, lib/, middleware/.
